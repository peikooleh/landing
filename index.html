О проекте

MoyaMova — приложение для запоминания иностранных слов. Сейчас доступны тренировки по немецкому и английскому на основе словарей из 2500+ слов уровней A1–B2.

Приложение бесплатное, не содержит рекламы, не собирает данные и работает офлайн. Проект развивается постепенно — обратная связь пользователей напрямую влияет на то, каким он станет дальше.

Запустить тренажёр

Тренажёр открывается прямо в браузере. На компьютере он запускается в отдельном компактном окне.
